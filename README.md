Assignments from earlier courses: 6.1010 (Fundamentals of Programming) and 6.1210 (Introduction to Algorithms).

These projects explore interpreters, games, image/audio processing, graphs, and logical reasoning..

### audio_processing_lab.py
Performs basic audio processing tasks like filtering and convolution.

### autocomplete_lab.py
Implements an autocomplete system that suggests words based on partial input.

### bacon_number_lab.py
Uses graph search to compute Bacon numbers and connection paths between actors.

### image_processing_1_lab.py
Applies simple image operations such as blurring and edge detection.

### image_processing_2_lab.py
Builds more complex image effects using layered transformations.

### lisp_1_lab.py
Implements the basics of a Lisp/Scheme interpreter, including parsing and evaluation.

### lisp_2_lab.py
Practices recursive list processing and functional-style programming patterns.

### minesweeper_lab.py
Implements the core logic of the Minesweeper game, including recursive reveals.

### recipes_lab.py
Calculates the cheapest way to make recipes by expanding ingredient dependencies.

### sat_lab.py
Solves Boolean satisfiability problems using recursive backtracking.

snekoban_lab.py
Simulates a Sokoban-style puzzle game with movement rules and win conditions.

symbolic_algebra.py
Builds a symbolic math system that can simplify expressions and take derivatives.
